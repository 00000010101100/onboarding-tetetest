# Staix Onboarding

Staix 온보딩 채팅 애플리케이션입니다.

## 기능

- 💬 실시간 채팅 인터페이스
- 🔄 웹훅 API 연동
- 📱 반응형 디자인 (최소화/최대화 모드)
- 🎨 Staix 브랜드 디자인 시스템
- 🔗 자동 URL 링크 변환
- 📷 조건부 이미지 표시

## 기술 스택

- React 18
- TypeScript
- Vite
- Tailwind CSS
- ShadCN UI

## 로컬 개발

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev
```

## 배포

Vercel을 통한 자동 배포가 설정되어 있습니다.
